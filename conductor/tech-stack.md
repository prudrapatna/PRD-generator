# Technology Stack

## 1. Programming Languages
*   **Python:** Used for scripting and potentially agent logic execution.
*   **Markdown:** Used for documentation, agent skills definition, and project artifacts.
*   **HTML:** Used for frontend structures.
*   **CSS:** Used for styling.

## 2. Frameworks & Libraries
*   **Agent Skills:** The core "framework" consists of specialized agent skills defined in Markdown.

## 3. Database & Storage
*   **File-Based Structure:** Data and state are persisted using a structured file system (JSON for state, Markdown for content).

## 4. Infrastructure & Deployment
*   **Local Environment:** The agent operates within a local development environment.
*   **Google Cloud Platform (GCP):** Target deployment environment.
